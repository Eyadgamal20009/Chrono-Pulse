## ğŸ› ï¸ How It Works
The system uses a laser diode and receiver harvested from waste electronics, controlled by a recycled ATmega328P microcontroller. 
1. **Laser Source:** Emits a continuous beam.
2. **Receiver:** Detects beam interruptions.
3. **Microcontroller:** Calculates time/velocity with microsecond precision.
4. **Web Interface:** This repository hosts the presentation interface, showcasing the methodology, results, and team info.
